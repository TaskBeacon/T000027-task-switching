# Task Plot Brief

## Task

- Title: Task Switching Task
- Construct: cognitive flexibility / task switching / switch cost
- Paradigm: cued task-switching with parity and magnitude rules.

## Rows

- Start: first trial or no previous rule.
- Repeat: current rule repeats the previous rule.
- Switch: current rule differs from the previous rule.

## Trial Timeline

1. Fixation: 300-600 ms jittered fixation, no response.
2. Cue: 600 ms. Show active rule cue and transition tag.
3. Decision: 2000 ms. Show target digit. Press F/J under the active rule.
4. Feedback: 800 ms. Show correct, incorrect, or timeout feedback and score update.
5. ITI: 300-600 ms jittered fixation, no response.

## Notes

- Rules are parity and magnitude.
- Parity mapping: F=odd, J=even.
- Magnitude mapping: F=less than 5, J=greater than 5.
- Target digits come from 1, 2, 3, 4, 6, 7, 8, 9.
